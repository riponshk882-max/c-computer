#### Preview

 - [Demo](https://themewagon.github.io/tailone/)

#### Download
 - [Download from ThemeWagon](https://themewagon.com/themes/tailone/)

## Getting Started

1. Clone Repository
```
git clone https://github.com/themewagon/tailone.git
```
2. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```
